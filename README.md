oryol-test-app
==============

Oryol testing app in its own repo. To build and run from scratch:

```bash
> cd ~
> git clone git@github.com:floooh/oryol-test-app.git
> cd oryol-test-app
> ./fips build
> ./fips run TestApp
```

For more details, see the Oryol repository at https://github.com/floooh/oryol

Enjoy!
